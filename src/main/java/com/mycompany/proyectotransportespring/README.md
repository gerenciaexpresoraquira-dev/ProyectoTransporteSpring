# Proyecto Expreso Ráquira

## Descripción

Sistema web de reservas de transporte desarrollado con Java y Spring Boot.

El sistema permite registrar reservas de pasajeros para rutas de transporte, generando confirmación automática y número de reserva.

---

## Tecnologías utilizadas

* Java 17
* Spring Boot
* Thymeleaf
* HTML5
* CSS3
* Maven
* NetBeans IDE

---

## Funcionalidades

* Registro de reservas
* Validación de formulario
* Generación automática de número de reserva
* Confirmación de reserva
* Interfaz gráfica moderna

---

## Estructura del proyecto

* Controller
* Templates HTML
* Configuración Spring Boot
* Archivo application.properties

---

## Framework utilizado

Spring Boot 3.3.0
